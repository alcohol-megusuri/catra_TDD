# 閉区間

閉区間を求める。  
例えば、`[3,8]` だったら `[3,4,5,6,7,8]` になる。
戻り値は string or list

### メモ

戻り値は数字の配列。

### 要件

- lower と upper を持つことができ、それらの閉区間を表す
  - 閉区間を返すことができる
  - 文字列表現ができる
  - lower > upper の場合は例外
  - 指定した整数を含むかどうかを判定できる
    - true or false
  - 別の閉区間と等価かどうかを判定できる
    - true or false
  - 別の閉区間に含まれるかどうかを判定できる
    - true or false
